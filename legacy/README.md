# Farhan Solaiman — Developer Portfolio 2026

Personal portfolio site for **Farhan Solaiman**, Senior Full Stack Developer & AI Automation Tech Lead.

**[View Live Site](https://farhansolaiman.github.io/farhan-solaiman-developer-portfolio-2026/)**

---

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, fluid typography (`clamp()`), 12-column grid, CSS animations
- **Vanilla JavaScript** — No frameworks or dependencies
- **Google Fonts** — Space Grotesk, Inter, JetBrains Mono

## Features

- Responsive, mobile-first design
- Custom page loader with hero text reveal animation
- Scroll-reveal animations via IntersectionObserver
- Smooth scroll navigation with scroll spy
- Animated marquee banner
- Reduced motion support for accessibility
- Keyboard-accessible mobile menu
